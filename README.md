# 🕌 MesjidKu

Aplikasi **informasi dan manajemen masjid** yang menyediakan jadwal sholat, pengumuman kegiatan, dan informasi profil masjid.

## ✨ Fitur

- **Jadwal Sholat** — Informasi waktu sholat
- **Profil Masjid** — Detail dan galeri masjid
- **Kegiatan** — Informasi acara masjid
- **Pengumuman** — Berita terbaru seputar masjid

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
